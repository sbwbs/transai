## Key Concepts

### Translation Patterns
- **Definition:** Recurring characteristics in successful/unsuccessful translations
- **Examples:** Sentence length, terminology usage, formatting
- **Applications:** Prompt optimization, quality improvement

### Protocol Pairing
- **Definition:** Using pairs of related medical protocols for translation context
- **Benefits:** Better understanding, higher accuracy
- **Challenges:** Requires protocol library, matching logic

### Feedback Analysis
- **Definition:** Systematic analysis of translation feedback
- **Process:** Collect → Categorize → Analyze → Recommend → Implement
- **Outcomes:** Continuous improvement, error reduction
